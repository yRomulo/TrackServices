# Track Services Monorepo

Estrutura inicial para Fase 1 do plano de acao com foco no frontend.

## Estrutura

- `apps/frontend`: frontend com Next.js + Tailwind CSS
- `packages`: espaco para bibliotecas compartilhadas futuras

## Como rodar

1. Instale dependencias na raiz:
   - `npm install`
2. Rode o frontend:
   - `npm run dev:frontend`
3. Acesse:
   - `http://localhost:3000`
